# 每日一字功能改造计划

## 一、页面结构调整

1. **首页布局学习菜根谭摘录风格**
   - 采用顶部导航+主内容区+底部操作栏的结构
   - 顶部添加保存、收藏、测试三个功能按钮
   - 底部添加上一字、下一字按钮
   - 移除原有学习字卡和开始测试按钮

2. **字卡内容调整**
   - 移除常见错误区域
   - 放大正确读音字体，使用醒目颜色
   - 优化相关词组显示，字体放大
   - 实现字卡点击翻转效果

3. **字卡背面设计**
   - 上部显示缩小版正面内容
   - 下部依次显示：详细解析、常见错误读音、错误原因分类、相关古诗词

4. **难度模式选择**
   - 添加初级、中级、高级、挑战级四种难度选择
   - 支持多选，默认全选
   - 点击切换选择状态
   - 根据选择过滤显示的字卡

## 二、功能实现

1. **字卡翻转动画**
   - 点击字卡正面翻转到背面
   - 点击字卡背面返回正面
   - 实现平滑的3D翻转效果

2. **随机翻阅功能**
   - 上一字、下一字按钮实现随机切换字卡
   - 支持根据难度过滤后的随机顺序

3. **保存功能**
   - 实现保存字卡图像到相册
   - 生成包含正面大字、背面小字、小程序码和分享说明的图像

4. **收藏功能**
   - 本地存储用户收藏的字列表
   - 支持添加/取消收藏
   - 收藏状态实时更新

5. **测试功能**
   - 点击顶部测试按钮进入测试页面
   - 传递当前字或随机字到测试页面

## 三、样式优化

1. **整体风格**
   - 学习菜根谭摘录的典雅风格
   - 采用暖色调配色方案
   - 优化字体大小和间距

2. **字卡设计**
   - 正面突出字和正确读音
   - 背面信息层次分明
   - 添加卡片阴影和圆角
   - 实现流畅的翻转动画

3. **按钮设计**
   - 统一按钮样式
   - 添加点击反馈效果
   - 优化按钮布局

## 四、实施步骤

1. **更新入口页面WXML结构**
   - 调整整体布局
   - 添加顶部功能按钮
   - 改进字卡显示
   - 添加底部操作按钮
   - 添加难度选择器

2. **实现字卡翻转功能**
   - 添加翻转动画CSS
   - 实现翻转逻辑
   - 设计正面和背面内容

3. **更新JavaScript逻辑**
   - 实现上一字、下一字功能
   - 实现难度过滤
   - 实现收藏功能
   - 实现保存功能占位

4. **优化样式**
   - 更新CSS样式
   - 实现平滑动画
   - 确保响应式设计

5. **测试与调试**
   - 测试所有功能
   - 调试样式问题
   - 确保兼容性

## 五、预期效果

1. 页面布局清晰美观，符合菜根谭摘录的典雅风格
2. 字卡翻转效果流畅自然
3. 难度选择功能正常工作
4. 收藏和保存功能实现
5. 整体用户体验良好

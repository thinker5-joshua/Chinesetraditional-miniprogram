# 分组看页面布局优化方案

## 问题分析
1. **模式按钮布局不合理**：当前顺序和八宫两个模式按钮居中显示在一行，但不是分左右两边
2. **卦卡片高度过高**：当前卦卡片高度为150px，占用过多空间
3. **组件间距过大**：各组件间的上下间距过大，导致页面无法在一屏中显示
4. **整体布局松散**：页面整体布局不够紧凑，存在大量空白区域

## 解决方案

### 1. 优化模式按钮布局
- 将顺序和八宫两个模式按钮分左右两边显示
- 修改 `.mode-buttons` 的 `justify-content` 为 `space-between`
- 调整 `.mode-selector` 的 `flex-direction` 为 `row`
- 移除不必要的内边距和外边距

### 2. 降低卦卡片高度
- 将卦卡片高度从150px降低到120px
- 调整卡片内部元素的字体大小和间距
- 优化卡片内边距，从12px调整到8px

### 3. 紧凑各组件间距
- 减小header下方margin-bottom从10px到5px
- 减小mode-selector下方margin-bottom从15px到8px
- 减小group-info下方margin-bottom从15px到8px
- 减小hexagram-grid下方margin-bottom从15px到8px
- 减小pagination上方margin-top从15px到8px
- 减小back-button-container上下margin从20px到10px

### 4. 优化字体大小和内边距
- 调整标题和副标题的字体大小
- 优化按钮内边距
- 调整分页控件的大小和间距

## 预期效果
1. 顺序和八宫两个模式按钮分左右两边显示
2. 卦卡片高度降低，显示更紧凑
3. 各组件间的间距减小，整体布局更紧凑
4. 整个页面能在一屏中显示
5. 保持良好的视觉层次和可读性

- **模式按钮布局**：修改为 `flex-direction: row` 和 `justify-content: space-between`，使按钮分左右两边显示
- **卦卡片高度**：从150px降低到120px，减小垂直空间占用
- **组件间距**：普遍减小5-12px，使整体布局更紧凑
- **内边距优化**：减小各组件内边距，进一步节省空间

这个方案将使分组看页面更加紧凑，能够在一屏中显示完整内容，同时保持良好的视觉效果和用户体验。

# 修改每日一字字卡布局方案

## 目标
1. 删除字卡背面的相关诗词部分
2. 增大其他部分的字体大小
3. 调整整体布局，使内容更加美观

## 实施步骤

### 1. 删除相关诗词部分
- 打开 `subgames/daily-character/pages/card/index.wxml` 文件
- 删除 `<view class="poems-section">` 部分及其所有子元素
- 打开 `subgames/daily-character/pages/card/index.wxss` 文件
- 删除 `.poems-section` 相关的样式定义

### 2. 增大字体大小
- 增大详细解析区的字体：将 `.explanation` 类的 `font-size` 从 28rpx 增大到 32rpx
- 增大相关词组的字体：将 `.phrase-item` 类的 `font-size` 从 26rpx 增大到 30rpx
- 增大易读错词组的字体：将 `.wrong-phrase-item` 类的 `font-size` 从 28rpx 增大到 32rpx
- 增大发音部分的字体：将 `.correct-pronunciation` 类的 `font-size` 从 36rpx 增大到 40rpx
- 增大 `.wrong-pronunciation` 类的 `font-size` 从 28rpx 增大到 32rpx

### 3. 调整整体布局
- 增加各区块之间的间距：将通用区块的 `margin-bottom` 从 40rpx 增加到 50rpx
- 增加卡片的内边距：将 `.card-container` 类的 `padding` 从 30rpx 增加到 40rpx
- 增加汉字与发音部分之间的间距：将 `.character` 类的 `margin-bottom` 从 30rpx 增加到 40rpx
- 调整词组网格的间距：增大 `.phrase-item` 类的 `padding` 从 15rpx 25rpx 增加到 20rpx 30rpx

### 4. 验证修改结果
- 运行小程序，查看修改后的字卡布局
- 确认相关诗词部分已被删除
- 确认其他部分的字体已增大
- 确认整体布局更加美观、合理

## 技术实现
- 直接修改 `subgames/daily-character/pages/card/index.wxml` 和 `subgames/daily-character/pages/card/index.wxss` 文件
- 删除不需要的诗词部分
- 调整字体大小和布局参数
- 确保修改后的布局在不同屏幕尺寸下都能正常显示

## 预期成果
- 字卡背面不再显示相关诗词部分
- 其他部分的字体更大，更易于阅读
- 整体布局更加美观、合理，充分利用了删除诗词部分后腾出的空间

# 六爻按钮样式进一步优化方案

## 问题分析
1. **按钮距离边框的左右间距较大**：当前容器内边距较大，导致按钮距离边框较远
2. **按钮之间的间距可以适当增大**：当前按钮间距在不同屏幕尺寸下分别为10px、8px、6px，可以进一步增大
3. **按钮宽度可以适当加宽**：当前按钮最小宽度在不同屏幕尺寸下分别为25px、30px、25px，可以适当加宽

## 解决方案

### 1. 优化默认样式（.six-yao-buttons）
- 减小容器内边距：将 `padding` 从 `30px` 减小到 `20px`，减小按钮距离边框的左右间距
- 增大按钮间距：将 `gap` 从 `10px` 增大到 `15px`，增大按钮之间的间距
- 保持 `width: calc(100% - 30px)` 不变，确保与上部卦卡片对齐

### 2. 优化默认按钮样式（.yao-button）
- 加宽按钮宽度：将 `min-width` 从 `25px` 增大到 `40px`，加宽按钮宽度
- 保持 `padding: 8px 5px` 不变，确保按钮内容清晰可见
- 保持 `gap: 3px` 不变，确保按钮内部元素间距合理

### 3. 优化平板响应式样式（@media (max-width: 768px)）
- 调整 `.six-yao-buttons`：
  - 减小容器内边距：将 `padding` 从 `20px` 减小到 `15px`
  - 增大按钮间距：将 `gap` 从 `8px` 增大到 `12px`
- 调整 `.yao-button`：
  - 加宽按钮宽度：将 `min-width` 从 `30px` 增大到 `35px`
  - 保持 `padding: 6px 6px` 不变，确保按钮内容清晰可见

### 4. 优化手机响应式样式（@media (max-width: 480px)）
- 调整 `.six-yao-buttons`：
  - 减小容器内边距：将 `padding` 从 `15px` 减小到 `10px`
  - 增大按钮间距：将 `gap` 从 `6px` 增大到 `10px`
- 调整 `.yao-button`：
  - 加宽按钮宽度：将 `min-width` 从 `25px` 增大到 `30px`
  - 保持 `padding: 5px 5px` 不变，确保按钮内容清晰可见

## 预期效果
1. 按钮距离边框的左右间距减小，使整体布局更加紧凑
2. 按钮之间的间距增大，使按钮之间的区分更加清晰
3. 按钮宽度加宽，使按钮内容更加清晰可见
4. 在所有屏幕尺寸下都能保持良好的显示效果
5. 与上部卦卡片保持对齐，整体布局协调

这个方案将进一步优化六爻按钮的布局和样式，使按钮距离边框的左右间距减小，按钮之间的间距增大，按钮宽度加宽，整体布局更加紧凑和美观。

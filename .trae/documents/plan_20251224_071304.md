# 卦卡片绘制优化方案

## 问题分析
1. 当前卦象、卦名和拼音的布局不符合用户要求
2. 用户希望将三者分为左右两个区绘制，左区放卦象，右区放卦名和拼音
3. 卦名和拼音需要分两行，卦名在上，拼音在下
4. 各区域内容需要居中对齐

## 解决方案

### 1. 分区绘制方案
- 将画布宽度分为左右两个区，各占一半
- 左区：绘制卦象符号，居中对齐
- 右区：绘制卦名和拼音，卦名在上，拼音在下，均居中对齐
- 计算分区边界：midX = width / 2

### 2. 具体绘制步骤
- 左区绘制卦象：位置为 midX / 2, 350，居中对齐
- 右区绘制卦名：位置为 midX + (midX / 2), 320，居中对齐
- 右区绘制拼音：位置为 midX + (midX / 2), 370，居中对齐
- 调整字体大小和行高，确保整体布局协调

### 3. 优化绘制位置和字体大小
- 卦象字体大小保持120px
- 卦名字体大小保持48px
- 拼音字体大小保持32px
- 调整绘制位置，使整体布局更协调

## 预期效果
1. 卦象、卦名和拼音按照左右分区方式绘制
2. 左区卦象居中对齐
3. 右区卦名在上，拼音在下，均居中对齐
4. 整体布局更清晰、美观

这个方案将优化卦卡片的绘制，使布局更符合用户要求，同时保持内容清晰可见。

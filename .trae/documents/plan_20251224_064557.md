# 卦卡片绘制优化方案

## 问题分析
1. **卦排名位置和字体大小**：当前绘制在中间，字体大小为36px，需要调整到右上角，字体更小
2. **卦名和拼音布局**：当前分别绘制在两行，需要调整到一行，中间适当间距
3. **卦别名和上下卦布局**：当前分别绘制在两行，需要调整到一行，中间适当间距
4. **爻辞内容字体大小**：当前字体大小为34px，需要改小以容纳更多内容
5. **爻辞行间距问题**：一爻和二爻之间多了个空行，需要检查并修复

## 解决方案

### 1. 优化卦排名绘制
- 将卦排名绘制在卡片右上角
- 减小字体大小到28px
- 将文本对齐方式改为右对齐
- 调整绘制位置到右上角

### 2. 优化卦名和拼音布局
- 将卦名和拼音绘制在一行
- 卦名字体大小保持48px
- 拼音字体大小保持32px
- 中间添加适当间距（约20px）
- 先绘制卦名，然后在卦名右侧绘制拼音

### 3. 优化卦别名和上下卦布局
- 将卦别名和上下卦绘制在一行
- 卦别名保持36px
- 上下卦保持32px
- 中间添加适当间距（约20px）
- 先绘制卦别名，然后在右侧绘制上下卦

### 4. 优化爻辞内容字体大小
- 将爻辞内容字体大小从34px减小到28px
- 保持行高适当，确保内容清晰可见

### 5. 修复爻辞行间距问题
- 检查并修复一爻和二爻之间多了个空行的问题
- 问题原因：在绘制完一爻后，startY增加了lineHeight，而在绘制二爻时，又从startY + i * lineHeight开始绘制，导致重复增加lineHeight
- 解决方案：移除多余的startY增加，或者调整lineY的计算方式

## 预期效果
1. 卦排名绘制在卡片右上角，字体更小，不占用主要视觉空间
2. 卦名和拼音在一行，卦别名和上下卦在一行，布局更紧凑
3. 爻辞内容字体更小，能容纳更多内容
4. 一爻和二爻之间的空行问题被修复，行间距均匀

# 六爻按钮样式优化方案

## 问题分析
1. **响应式样式不合理**：在小屏幕上，按钮的 `min-width` 反而比大屏幕大，导致按钮挤在一起
2. **按钮间距过小**：在小屏幕上，按钮间距仅为 2-3px，导致按钮挨得太近
3. **容器内边距不足**：在小屏幕上，容器内边距太小，导致按钮挤到边框
4. **样式不统一**：不同屏幕尺寸下的样式设置不一致，导致显示效果不佳

## 解决方案

### 1. 优化默认样式（.six-yao-buttons）
- 保持 `padding: 30px` 不变，确保边框高度合适
- 保持 `gap: 10px` 不变，确保按钮间距合理
- 保持 `width: calc(100% - 30px)` 不变，确保与上部卦卡片对齐

### 2. 优化默认按钮样式（.yao-button）
- 保持 `padding: 8px 5px` 不变，确保按钮内容清晰可见
- 保持 `min-width: 25px` 不变，确保按钮宽度合适
- 保持 `gap: 3px` 不变，确保按钮内部元素间距合理

### 3. 优化平板响应式样式（@media (max-width: 768px)）
- 调整 `.six-yao-buttons`：
  - `padding: 20px`（适当减小内边距，但保持足够高度）
  - `gap: 8px`（适当减小间距，但保持按钮清晰分开）
- 调整 `.yao-button`：
  - `padding: 6px 6px`（适当减小内边距）
  - `min-width: 30px`（适当减小宽度，确保在平板上能完整显示）
  - `gap: 2px`（适当减小内部间距）

### 4. 优化手机响应式样式（@media (max-width: 480px)）
- 调整 `.six-yao-buttons`：
  - `padding: 15px`（进一步减小内边距，但保持合理高度）
  - `gap: 6px`（进一步减小间距，但保持按钮清晰分开）
- 调整 `.yao-button`：
  - `padding: 5px 5px`（进一步减小内边距）
  - `min-width: 25px`（确保在手机上能完整显示6个按钮）
  - `gap: 2px`（保持合理的内部间距）

### 5. 优化按钮内部元素样式
- 确保 `.yao-text` 和 `.yao-symbol` 在所有屏幕尺寸下都有合适的字体大小
- 保持字体加粗效果，确保六爻字体清晰可见

## 预期效果
1. 六爻按钮在所有屏幕尺寸下都能完整清晰地显示在一屏内
2. 按钮间距合理，不会挤在一起
3. 按钮宽度适中，不会过长或过短
4. 按钮内容清晰可见，字体大小合适
5. 容器边框高度合适，内边距充足
6. 与上部卦卡片对齐，整体布局协调

这个方案将彻底解决六爻按钮挤到边框、互相之间挨得太近的问题，确保在所有设备上都有良好的显示效果。

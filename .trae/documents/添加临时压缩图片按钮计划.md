# 添加临时压缩图片按钮计划

## 方案概述

在守护神兽首页添加一个临时按钮，用于导航到图片压缩页面。按钮将添加在合适的位置，并使用现有的样式，添加明确的注释说明这是临时按钮。

## 实现步骤

1. **修改首页WXML**：在首页底部添加一个临时按钮，用于进入压缩图片页面
2. **修改首页JS**：添加一个`goToCompressImages`方法，用于导航到压缩图片页面
3. **样式调整**：使用现有的按钮样式，确保按钮与整体页面风格一致
4. **添加注释**：为按钮添加明确的注释，说明这是临时按钮，后续会删除

## 具体实现

### 1. 修改首页WXML

在首页的`beasts-list-section`下方添加一个临时按钮：

```wxml
<!-- 临时压缩图片按钮，后续删除 -->
<view class="temp-button-section">
  <button class="temp-compress-btn" bindtap="goToCompressImages">
    临时进入图片压缩页面
  </button>
</view>
```

### 2. 修改首页JS

在首页的JS文件中添加`goToCompressImages`方法：

```javascript
/**
 * 临时方法：进入图片压缩页面
 */
goToCompressImages() {
  wx.navigateTo({
    url: '/subgames/guardian-beast/pages/compress-images/compress-images'
  });
}
```

### 3. 样式调整

在首页的WXSS文件中添加临时按钮的样式：

```css
/* 临时按钮样式，后续删除 */
.temp-button-section {
  display: flex;
  justify-content: center;
  margin: 40rpx 0;
}

.temp-compress-btn {
  background: rgba(255, 255, 255, 0.1);
  color: #64ffda;
  border: 2rpx solid rgba(100, 255, 218, 0.3);
  border-radius: 40rpx;
  padding: 20rpx 40rpx;
  font-size: 28rpx;
  font-weight: bold;
  box-shadow: 0 5rpx 15rpx rgba(100, 255, 218, 0.2);
  transition: all 0.3s ease;
}

.temp-compress-btn:active {
  background: rgba(100, 255, 218, 0.2);
  box-shadow: 0 3rpx 10rpx rgba(100, 255, 218, 0.3);
}
```

## 预期效果

- 在守护神兽首页底部添加一个临时按钮
- 按钮文字为"临时进入图片压缩页面"
- 点击按钮可以导航到压缩图片页面
- 按钮样式与整体页面风格一致
- 按钮有明确的注释，说明是临时按钮

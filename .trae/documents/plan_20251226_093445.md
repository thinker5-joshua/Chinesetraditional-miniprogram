# 调整字卡布局和样式

**问题**：
1. 相关词组需要往下一些
2. 字卡背面段落前有棕色线条，需要去除
3. 测试错误原因分类显示
4. 相关古诗词内容字体需要加大

**解决方案**：

1. **调整相关词组位置**：
   - 增加 `.related-phrases` 的 `margin-top` 从45rpx到60rpx，让相关词组往下一些

2. **去除段落前的棕色线条**：
   - 移除 `.detail-item` 的 `border-left: 5rpx solid #8B4513`
   - 移除 `.detail-title` 的 `border-left: 5rpx solid #8B4513`
   - 移除 `.poem-item` 的 `border-left: 4rpx solid #CD853F`

3. **测试错误原因分类显示**：
   - 修改WXML中错误原因分类部分，将 `{{getErrorReasonText(todayCharacter.errorReasonType)}}` 替换为 "测试"

4. **加大古诗词字体**：
   - 将 `.poem-line` 的字体大小从24rpx增加到28rpx
   - 将 `.poem-author` 的字体大小从22rpx增加到26rpx

**预期效果**：
- 相关词组位置下移，布局更合理
- 字卡背面段落前的棕色线条被移除，界面更简洁
- 错误原因分类直接显示"测试"，便于调试
- 相关古诗词内容字体加大，更易阅读

# 测试页相关词组和提示语间距调整方案

## 问题分析

测试页中相关词组和下方提示语的间距需要拉大一些。通过分析代码结构：

- 相关词组使用 `.related-phrases` 类，当前 margin 为 `20rpx 0`
- 提示语使用 `.question-prompt` 类，当前无 margin-top 设置
- 两者都在 `.question-section` 容器中，使用 flex 布局

## 修复方案

修改 `/subgames/daily-character/pages/test/index.wxss` 文件中的 `.related-phrases` 类，增加 margin-bottom 值：

```wxss
/* 相关词组样式 */
.related-phrases {
  margin: 20rpx 0 40rpx 0; /* 增加底部间距到40rpx */
  text-align: center;
  line-height: 1.5;
}
```

## 预期效果

- 相关词组和下方提示语之间的间距从原来的 20rpx 增加到 40rpx
- 页面布局更加清晰，各元素之间的层次更加分明
- 保持其他样式不变，确保整体视觉效果协调

## 实现步骤

1. 打开 `/subgames/daily-character/pages/test/index.wxss` 文件
2. 找到 `.related-phrases` 类的样式定义
3. 修改 margin 值，将底部间距从 20rpx 增加到 40rpx
4. 保存文件并在微信开发者工具中预览效果
5. 确认间距调整合适后完成修复

# 每日一字功能优化计划

## 一、优化内容

### 1. 难度模式优化
- 移除"难度模式"字眼
- 用框框住四个难度模式选项
- 实现点亮/未点亮两种状态，通过背景色和字体颜色变化
- 记住上次选择的难度模式，下次进入时自动应用

### 2. 移除"点击卡片查看详情"提示

### 3. 字卡布局调整
- 解决字卡右边冲出屏幕的问题
- 字卡整体向上、向左移动，减少左边和上边的空间
- 适当加宽字卡宽度

### 4. 上一字、下一字按钮优化
- 增加按钮长度，避免文字折行

### 5. 间距和布局调整
- 增加相关词组与上方的间距
- 固定上一字、下一字和难度模式框在屏幕下方
- 固定字卡高度，正面排版美观
- 背面内容尽量在高度内完全显示，超出部分滚动查看

### 6. 字卡背面优化
- 移除字和正确读音下面的框或背景色
- 详细解析、常见错误读音、错误原因分类前面的说明文字后加冒号，内容紧接在后，尽量一行显示

## 二、修改计划

1. **更新WXML结构**
   - 调整字卡容器的布局
   - 修改难度模式选择器的结构
   - 移除"点击卡片查看详情"提示
   - 调整上一字、下一字按钮的样式

2. **优化CSS样式**
   - 调整字卡的位置和宽度
   - 修改难度模式选择器的样式，实现点亮/未点亮状态
   - 增加上一字、下一字按钮的长度
   - 增加相关词组与上方的间距
   - 固定底部操作栏在屏幕下方
   - 固定字卡高度
   - 优化字卡背面的样式

3. **更新JavaScript逻辑**
   - 保存和读取上次选择的难度模式
   - 优化字卡的定位和布局
   - 调整字卡背面的内容排版

## 三、预期效果

1. 难度模式选择更直观，状态清晰，记住用户选择
2. 字卡布局更合理，避免冲出屏幕，位置居中
3. 按钮文字不再折行，美观易用
4. 整体布局更紧凑，空间利用更合理
5. 字卡背面内容排版更简洁，信息层次更清晰

## 四、实施步骤

1. 更新入口页面WXML结构
2. 优化CSS样式，调整布局和样式
3. 更新JavaScript逻辑，实现记住难度模式选择
4. 测试所有功能，确保正常运行
5. 调整细节，优化用户体验

# 难读字数据收集与整理计划

## 一、项目目标
创建一个名为 `data-diff.js` 的数据文件，用于收集和整理以下类型的难读字：
1. 易读错的姓名中的汉字
2. 易读错的地名中的汉字
3. 生僻字（不常见的汉字）
4. 笔画特别多的汉字

## 二、数据结构
采用与 `data-optimized.js` 完全一致的数据结构：
```javascript
{
  id: 字ID,
  char: 汉字,
  correctPronunciation: 正确读音,
  wrongPronunciations: 错误读音列表,
  relatedPhrases: 关联词组,
  explanation: 字典解释,
  errorReasonType: 常见错误原因分类,
  difficultyLevel: 测试难度等级（初级、中级、高级、挑战级）
}
```

## 三、数据来源与收集方法

### 1. 已提供的图片数据
从用户提供的图片中提取易读错字：

#### 易读错地名（部分示例）：
- 崔家岙 (Cuī jiā ào) - 岙 ào
- 酂阳 (Cuó yáng) - 酂 Cuó
- 大埔 (Dà bù) - 埔 bù
- 柘 (Zhè) - 柘 Zhè
- 訾 (Zī) - 訾 Zī
- 钟馗 (Zhōng kuí) - 馗 kuí
- 珲春 (Hún chūn) - 珲 Hún
- 百家疃 (Bǎi jiā tuǎn) - 疃 tuǎn
- 百灵淖 (Bǎi líng nào) - 淖 nào
- 蚌埠 (Bèng bù) - 蚌埠 Bèng bù

#### 易读错姓名（部分示例）：
- 柘 Zhè
- 訾 Zī
- 钟馗 Zhōng kuí

### 2. 生僻字与多笔画字收集
- 生僻字：如 龘(dá)、靐(bìng)、齉(nàng) 等
- 多笔画字：如 齉(nàng)、龘(dá)、龖(dá) 等

## 四、实施步骤

### 1. 创建初始数据文件
- 创建 `data-diff.js` 文件，复制 `data-optimized.js` 的结构
- 初始化文件头注释和数据数组

### 2. 提取现有图片数据
- 从用户提供的三张图片中提取所有易读错字
- 按照数据结构整理每个字的信息
- 确保每个字包含正确读音、常见错误读音、关联词组等信息

### 3. 补充生僻字和多笔画字
- 收集常见的生僻字和多笔画字
- 为每个字添加完整的释义和相关信息
- 确保数据的准确性和完整性

### 4. 数据验证与优化
- 检查每个字的读音和释义是否正确
- 确保数据结构与 `data-optimized.js` 完全一致
- 优化数据组织，便于后续导入云数据库

## 五、预期成果
- 一个完整的 `data-diff.js` 文件，包含所有收集的难读字
- 数据结构与现有数据文件一致，便于后续处理
- 为后续使用大模型或人工完善数据奠定基础
- 为导入云数据库做好准备，可直接用于每日一字功能

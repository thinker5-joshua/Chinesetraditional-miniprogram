# 六十四卦学习优化方案（最终调整）

## 一、核心优化点
1. **模式选择按钮边距优化**：调整边框宽度、内边距、外边距和按钮宽度
2. **卦别名和上下卦背景色**：移除背景色
3. **六爻按钮优化**：字体加粗、减小宽度、调整间距

## 二、详细实现方案

### 1. 模式选择按钮边距优化
- **修改文件**：`subgames/64Hexagrams/index.wxss`
- **实现内容**：
  - 调整 `.mode-selector` 的 `width` 为 `calc(100% - 30px)`，与卦卡片宽度一致
  - 调整 `.mode-selector` 的 `padding` 为 `15px 30px`，上下内边距减小，左右内边距保持30px
  - 调整 `.mode-buttons` 的 `gap` 为 `20px`
  - 调整 `.mode-button` 的 `width` 为 `100px`，确保不冲出边框
  - 调整 `.mode-button` 的 `margin` 为 `0`

### 2. 卦别名和上下卦背景色
- **修改文件**：`subgames/64Hexagrams/index.wxss`
- **实现内容**：
  - 移除 `.hexagram-extra-info` 的 `background-color` 属性
  - 调整 `.hexagram-extra-info` 的 `padding` 为 `0`

### 3. 六爻按钮优化
- **修改文件**：`subgames/64Hexagrams/index.wxss`
- **实现内容**：
  - 确保 `.yao-text` 和 `.yao-symbol` 的 `font-weight` 为 `bold`
  - 调整 `.yao-button` 的 `padding` 为 `3px 3px`，减小按钮宽度
  - 调整 `.six-yao-buttons` 的 `gap` 为 `5px`
  - 调整 `.six-yao-buttons` 的 `margin-top` 和 `margin-bottom` 为 `10px`，减小与卦卡片框和下方说明文字的间距
  - 调整 `.six-yao-buttons` 的 `padding` 为 `15px 10px`，加大与外部边框上下的间距

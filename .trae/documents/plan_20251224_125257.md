## 全面重写识卦页布局

### 修改目标
1. 让卦象列向左移动靠近左边框
2. 确保卦象列宽度为60px
3. 确保两列间距为20px
4. 确保卦名列宽度为300px

### 修改文件
- **matching.wxss** - 全面重写识卦页的布局样式

### 修改内容

#### 1. 重置游戏容器样式
- 移除 `.game-container` 中可能影响布局的复杂内边距设置
- 设置简单的内边距，确保内容可以靠近边框

#### 2. 重写游戏板容器样式
- 使用 `display: flex` 确保灵活布局
- 设置 `justify-content: flex-start` 让内容向左对齐
- 设置 `gap: 20px` 确保两列间距
- 移除不必要的padding和margin
- 使用 `box-sizing: border-box` 确保宽度计算正确

#### 3. 重写卦象列样式
- 设置 `width: 60px` 确保固定宽度
- 设置 `flex: 0 0 60px` 确保不伸缩
- 设置 `margin: 0` 和 `padding: 0` 移除额外间距
- 设置 `align-items: center` 确保内部内容居中
- 设置 `min-width: 60px` 和 `max-width: 60px` 锁定宽度

#### 4. 重写卦名列样式
- 设置 `width: 300px` 确保固定宽度
- 设置 `flex: 0 0 300px` 确保不伸缩
- 设置 `margin: 0` 和 `padding: 0` 移除额外间距
- 设置 `align-items: center` 确保内部内容居中
- 设置 `min-width: 300px` 和 `max-width: 300px` 锁定宽度

#### 5. 重写卦象方块样式
- 设置 `width: 100%` 确保继承父容器宽度
- 移除可能影响宽度的 `min-width` 或 `max-width` 设置
- 确保内部内容居中显示

#### 6. 移除冲突样式
- 检查并移除可能与布局冲突的其他样式
- 确保所有样式都简洁明了，没有冗余

### 预期效果
- 卦象列向左移动，靠近左边框
- 卦象列宽度准确显示为60px
- 两列间距保持20px
- 卦名列宽度准确显示为300px
- 整体布局简洁清晰，符合用户要求

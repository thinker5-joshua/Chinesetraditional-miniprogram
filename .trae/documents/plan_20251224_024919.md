# 六十四卦学习优化方案（按优先顺序）

## 一、核心优化点（按优先顺序）
1. **移除外部框**：移除包裹卦卡片、翻页按钮和六爻按钮的外部框，增加内部元素宽度
2. **卦别名和上下卦排版优化**：确保在小屏时也能一行显示
3. **六爻按钮布局优化**：确保在小屏时也能一行显示六个按钮
4. **阴爻按钮颜色调整**：调整为更淡的米色

## 二、详细实现方案

### 1. 移除外部框（优先）
- **修改文件**：`subgames/64Hexagrams/index.wxss`
- **实现内容**：
  - 移除 `.random-view` 的 `background-color`、`border-radius` 和 `box-shadow`
  - 移除 `.six-yao-view` 的 `background-color`、`border-radius` 和 `box-shadow`
  - 调整 `.random-card`、`.random-pagination` 和 `.six-yao-buttons` 的宽度
  - 移除 `.random-view` 和 `.six-yao-view` 的 `padding`

### 2. 卦别名和上下卦排版优化
- **修改文件**：`subgames/64Hexagrams/index.wxss`
- **实现内容**：
  - 移除媒体查询（max-width: 480px）中 `.hexagram-extra-info` 的 `flex-direction: column` 设置
  - 确保始终使用 `flex-direction: row`
  - 调整 `.hexagram-extra-info` 的 `gap` 和 `padding`
  - 减小 `.hexagram-alias` 和 `.hexagram-trigrams` 的 `font-size`

### 3. 六爻按钮布局优化
- **修改文件**：`subgames/64Hexagrams/index.wxss`
- **实现内容**：
  - 在媒体查询（max-width: 768px）中进一步减小 `.yao-button` 的 `padding` 和 `min-width`
  - 在媒体查询（max-width: 480px）中进一步减小 `.yao-button` 的 `padding` 和 `min-width`
  - 减小 `.yao-text` 和 `.yao-symbol` 的 `font-size`
  - 确保六个按钮能在一行显示

### 4. 阴爻按钮颜色调整
- **修改文件**：`subgames/64Hexagrams/index.wxss`
- **实现内容**：
  - 修改 `.yao-button.yin` 的 `background-color` 为更淡的米色
  - 使用 `#f8f5e6` 或类似的淡米色

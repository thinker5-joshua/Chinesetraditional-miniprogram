- 如果音效文件不存在，游戏会自动使用振动反馈
- 振动效果：成功时轻微振动，失败时重度振动
- 音效播放失败时也会降级为振动

## 备用方案

如果无法添加音频文件，游戏已内置：
- **成功**: `wx.vibrateShort({ type: 'light' })` - 轻微振动
- **失败**: `wx.vibrateShort({ type: 'heavy' })` - 重度振动
- **视觉反馈**: 成功时右下角闪烁显示配对的成语

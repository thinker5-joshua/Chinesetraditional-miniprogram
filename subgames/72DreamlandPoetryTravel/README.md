# 七十二洞天诗意旅行

## 游戏简介

七十二洞天诗意旅行是一款结合中国传统诗词文化与旅游胜地的教育游戏。玩家通过诗词挑战解锁中国各地的美景，体验诗意与自然的完美结合。

## 游戏特色

### 🏛️ 精选洞天
- 故宫博物院（北京）
- 黄山（安徽）
- 西湖（杭州）
- 泰山（山东）
- 长江三峡（重庆-湖北）
- 桂林山水（广西）

### 📜 诗词挑战
- 每个洞天配有相关的经典诗词
- 通过填字挑战测试诗词知识
- 答对后解锁洞天详情

### 🗺️ 地图功能
- 显示洞天地理位置
- 支持地图标记和点击交互
- 自动计算最佳视图范围

### 🎮 游戏流程
1. 选择洞天（点击地图上的红点）
2. 完成诗词填字挑战
3. 查看洞天详细介绍
4. 浏览相关诗词
5. 探索下一幻境

## 使用说明

### 启动游戏
1. 在小程序中进入"七十二洞天诗意旅行"
2. 查看星空背景下的洞天分布图
3. 点击"仙界任意门"开始探索

### 诗词挑战
1. 点击地图上的洞天红点
2. 仔细阅读诗词，找出缺失的字
3. 从选项中选择正确答案
4. 答对后进入洞天详情

### 探索洞天
1. 查看洞天介绍和地理位置
2. 浏览相关诗词作品
3. 点击"下一首"查看更多诗词
4. 选择"探索下一幻境"继续游戏

## 音效反馈

游戏配备音效系统：

- **振动反馈**：增强交互体验

## 开发说明

### 数据格式
洞天数据结构：
```javascript
{
  id: 数字ID,
  name: "洞天名称",
  location: "地理位置",
  description: "洞天介绍",
  coordinates: {
    latitude: 纬度,
    longitude: 经度
  },
  position: { x: 地图X%, y: 地图Y% },
  completed: 是否完成,
  relatedPoems: [相关诗词数组]
}
```

### 自定义诗词挑战
通过`poetry-data.js`中的`generatePoetryChallenge()`函数动态生成题目，支持：
- 随机选择诗词
- 随机隐藏字符
- 生成答案选项
- 智能排除重复
